Commands

All commands are prefixed with either '!' or '/'

  • User Commands

    • calladmin - Call an Admin to resolve issues
    • friendly - without the risk of death
    • rtd - Roll the dice

  • Voting Commands

    • votemenu - Menu with all vote options
    • valltalk - Toggle alltalk
    • vcrits - Toggle 100% crits
    • vmap - Vote to change the map
    • vrandom - Vote to toggle randomizer

  • Web Shortcuts

    • commands - Show this page in game
    • Discord - Show the discord invite
    • Donate - Every penny counts
    • Rules - They're important
